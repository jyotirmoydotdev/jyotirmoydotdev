## Task 0: Setup & Overview

In this task, you will prepare everything needed to start your Arduino journey.

- Understand what Arduino is
- Set up the Arduino IDE
- Connect your Arduino board to your computer
- Get ready for your first LED project

## What is Arduino?

Arduino is a small programmable board (microcontroller) that can control electronic components like LEDs, sensors, and motors.

Think of it as a **tiny computer** that you can program to interact with the real world.

## Install Arduino IDE

1. Go to the official Arduino website  
2. Download the Arduino IDE for your operating system  
3. Install it like any normal software  

## Connect Your Arduino

1. Plug your Arduino into your computer using a USB cable  
2. Open the Arduino IDE  
3. Go to:


```
Tools → Board → Select your Arduino board (e.g., Arduino Uno)

```

4. Then select the port:


```
Tools → Port → Select the correct COM/USB port

```

## Test Your Setup (Optional but Recommended)

Upload a basic example:

1. Go to:

```
File → Examples → Basics → Blink

```

2. Click **Upload**

If everything is correct, the built-in LED on your Arduino should start blinking

## Common Issues

- **Port not showing?** Try reconnecting the USB cable  
- **Upload failed?** Check if the correct board is selected  
- **LED not blinking?** Ensure proper connection
